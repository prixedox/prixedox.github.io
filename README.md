- [`/nn-playground/`](https://prixedox.github.io/nn-playground/) → repo `prixedox/nn-playground`
- [`/ray-tracer/`](https://prixedox.github.io/ray-tracer/) → repo `prixedox/ray-tracer`
- [`/bio-playground/`](https://prixedox.github.io/bio-playground/) → repo `prixedox/bio-playground`
- [`/shortcut-trainer/`](https://prixedox.github.io/shortcut-trainer/) → repo `prixedox/shortcut-trainer`

## Adding a new page

1. Create/publish the new project repo with GitHub Pages enabled
   (it will be served at `https://prixedox.github.io/<repo-name>/`).
2. In `index.html`, copy one `<li>` card block and update the
   `href`, title, and description.
3. Commit and push — the homepage updates automatically.
